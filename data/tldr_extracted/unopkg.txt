# unopkg


- Add and deploy given extension:

`unopkg add {{path/to/extension}}`

- Remove extension:

`unopkg remove {{extensions_id}}`

- Display information about deployed extensions:

`unopkg list`

- Raise extensions dialog (GUI):

`unopkg gui`

- Reinstall all deployed extensions:

`unopkg reinstall`

- Display help:

`unopkg {{[-h|--help]}}`
